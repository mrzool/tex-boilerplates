What is this for

TeX's purpose is to achieve the best possible automated typography. This project gets you started in seconds using tools you probably already know about. Forget about MS Word.

how does it work

The boilerplates leverage the powerful Pandoc to grab the content from YAML and Markdown files, populate a LaTeX template, and output a high-quality PDF through XƎTeX.

How to get
started

Each boilerplate is thoroughly documented. Pick a template above and check out the README for detailed instructions on how to get started.

Modern & Customizable

Use all your system fonts out of the box and adjust the margins just by changing one setting. There is even built-in support for a custom letterhead.
